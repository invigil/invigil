# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities privately to **security@linuxcent.com**.
Do not open a public issue for a security problem.

You'll get an acknowledgement within 72 hours and a remediation timeline once the
report is triaged. Coordinated disclosure is appreciated.

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |

Invigil is pre-1.0; only the latest minor line receives security fixes until a
1.0 release establishes a longer support window.
